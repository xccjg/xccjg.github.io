如何选择注塑机

每副模具只能装置在与其相顺应的注塑机上消费，才干稳定地消费出合格的制品，那么依据现有的模具来选择机台，或依据现有的机台来设计模具时，应细致地理解注塑机的技术标准，使之相配。
注塑模与注塑机需作以下三方面的校核：

A、工艺参数的校核，包括：

1、最大注塑量

2、最大注塑压力

3、最大锁模力

4、最大成型面积

5、塑化才能

B、装置尺寸的校核，包括：

1、最大模厚与最小模厚

2、模具外形尺寸的校核

3、模板装置模具的螺钉、钉孔（或T形槽）的位置和尺寸

4、注塑机喷嘴孔直径和喷嘴球头半径值

C、开模行程的校核

现引见如下：

一、工艺参数的校核

1、最大注塑量的校核

注塑重量是选择注塑机的最常用的一个参数，以安土（1安土=28.3克）或克表示。这个参数固然简单明了，但亦容易被误解。由于国产注塑机常用理论注塑容积来表示机器的最大注塑才能，需经换算才干得知最大注塑量，而实践注塑重量是丈量出来的。详细丈量如下：物料采用纯PS，把料筒加热到正常工艺温度，普通喷嘴温度比料筒温度的设定高出6℃，预塑时将喷嘴闭锁成无流延状态，并在额定注射压力下以最高注射速度和最大注射行程，连续对空注射三次，取其重量的均匀值，作为实践注射重的丈量值，丈量结果普通不得小于理论注塑量的95%。

（1）由于注塑机最大注塑量是实测对空注塑量，在注入模具时，由于活动阻力增加，加大了沿螺杆方向的逆流量，再思索平安系数， 实践量M1应取机器最大注塑才能的85%。

M1=85%MPSO

关于其它非PS之非结晶料，其最大注塑量：

M0=MPSOP/PPS

式中P——常温下某塑料的密度，g/Cm3

PPS——常温下PS的密度，g/Cm3

关于非结晶塑料能够为从常温状态到熔融状态，其密度变化倍率与PS变化陪率差不多，故可用常温下密度代入计算。而结晶形塑料由于从固态到熔态密度变化较PS变化更大。因而结晶塑料还要乘以一核正系数：

M0=0.9MPSOP/PPS

同理，其实践最大注射量M1=85%M0

例如：1）UPVC制品与浇道塑料共重4安土，物料为非结晶性料，密度为1.38g/CM3，请问需求注塑重量多大的注塑机？即M1=4zon，求MPSO？

MPSO = PPS * M O/ PPVC = POS / PPVC * M1 / 0.85 = 1.05 / 1.38 * 4 / 0.85 = 3.58（安土）

2）PA制件与浇道料共重4安土，物料为结晶性塑料，密度为1.15g/Cm3, 请问需求注塑重量多大的注塑机? 即M1=4zon，求Mpso？

MPSO = PPS / 0.9 / PPAB   * M1 / 0.85 = 1.05 / 0.9 / 1.15 * 4 / 0.85 = 4.77（安土）

（2）国产注塑机常用理论注塑容积VC表示机器的最大注塑才能，该体积是指在最大注塑行程时注塑螺杆所经过的最大致积，它与PS表示的最大注塑量的关系是：

MSO=VC * AS * PS

式中：P——为常温下塑料的密度

A——注塑系数，普通与塑料加温后体积收缩系数及螺杆逆流漏料有关

普通结晶料的体积收缩系数为0.85

非结晶料的体积收缩系数为0.93

思索到螺杆逆流漏料，结晶型塑料取A=0.7～0.8

非结晶型塑料(含PS)取A=0.8～0.9

（3）为确保塑件质量，注塑模一次成型的塑料质量（塑件和流道凝料质量之和）应在公称注塑量的30%～85%范围内。而产品请求不高，物料不易合成的状况下，最大可达90%，最小不小于10%亦可。但为了保证塑件质量，又充沛发挥设备的才能，选择范围通常在注塑量的50%～85%之间。

2、注射压力的校核

注塑压力校核是校验注塑机最大注塑压力能不能满足该制品成型的需求。制品成型所需的压力是由注塑机类型、喷嘴型式、塑料活动性、浇注系统和型腔的活动阻力等要素决议的。

为了保证产品的注塑质量，注塑时即便在高的压力下也需求一个十分稳定的注射速度（高质量注射），因而只要当所需压力不超越一定值（通常大约为最大压力的80%），液压系统的工作才干保证有相当稳定的注射性能（注射时螺杆向前运动的速度稳定）。而电机直接驱动安装高压注射时性能的降落以至可能比液压注射安装的还要大。

此外还必需思索到可用于注射的能量有5%～10%损耗在液压缸、油路中的磨擦上，关于不同的机器，这种能耗能肯定是不同的。因而，当注塑质量请求高的制品时，由于这种共同的特性，当同一套模具，改换同型号的机台时，也需重新调整工艺参数。

因而，依据注射压力选择机台时，可参考下列表中参数并应用下列方程停止计算

PMAX=1.25P

式中PMAX——机台最大注射压力

P——注塑制品所需注射压力

附：1）实践值不一定等于表中所列的最大压力

2）活动水平是从资料的流体特性、温度和对流体的阻力来定义

3、锁模力的校核

当高压的塑料熔体充溢型腔时，会在型腔内产生一个很大的力，力图使模具沿分型面涨开，其值等于塑件和流道系统在分型面上总投影面积乘以型腔内塑料压力。关于三板式模具或热流道模具，由于流道系统与型腔不在一个分型面上，则不应计入流道面积。作用在这个面积上的总力应小于注塑机的额定锁模力，否则在注塑时会因锁模不紧而产生溢边跑料的现象。型腔内塑料熔体的压力按下式计算：

P = K * PO

注塑机额定锁模力：

F = P * A

式中：P——模具型腔及流道内塑料熔体的均匀压力（Mpa）

P6——料筒内螺杆施于塑料熔体的压力（Mpa）

A——制件与浇注流道在分型面上的总投影面积（Cm2）

K——损耗系数   其随塑料种类、注塑机方式、喷嘴阻力、模具流道阻力而不同，普通在1/3~2/3范围内选择（螺杆式注塑机的K值比柱塞式大，直通喷嘴比弹簧喷嘴大）。

由于影响型腔压P与损耗系数的K的要素较复杂， 因而用通用塑料消费小型制品时，模腔压力常取20～40Mpa，即2000TON～4000TON/m2，而流程越长，壁越薄的塑件或活动性差以及加玻纤（GF）的原料，则需更大的锁模力。

如：PC制品，所需锁模力：4700TON～7700TON/m2（3-5吨/平方英寸）

PMMA制品，所需锁模力：3100TON～6200TON/m2（2-4吨/平方英寸）

PBT制品，所需锁模力：4700TON～7000TON/M2（3-4.5吨/平方英寸）
例如要注塑不断径79毫米的普通PS杯，此杯均匀厚度1mm，长径比为150：1，求足够的锁模力：

解：投影面积：A = Л * 792 / 4 = 49Cm2

查表可知： P = 70.6Mpa

则 F = 49 * 0.706 = 34.6ton

再例如：PC料，厚度平均3mm，直浇口投影面积A = 100cm2，求 F？

解：由于厚度为3mm，属厚度适中，取P0=120Mpa，K=1/2

则F = 100 * 120 * 0.5 = 60ton

计算出所需的锁模力后，可依据注塑机供给厂家的机器规格表，选用上一层次的机器。

4、塑化才能

依据制品消费周期、注塑容量与注塑机的塑化才能相匹配的准绳来选用机型，依据下式可求得塑化才能：

G = g * X / 1000

式中 G——塑化才能Kg/h

G——每模制件与浇口料的总重量（g）

X——每小时的注塑次数。

例如：一出四的热流道PET管坏模， 总重120g，注射周期为每分钟5次，计300模/小时，请问PT80B，ø35螺杆的注塑机适宜否（塑化方面）？

解：由规格表查得PT80B塑化才能G = 42 Kg/h

g * X / 1000=120 * 300 ÷ 1000 = 36 Kg/h < G

因而PT80B是适宜的。

二、模具外形尺寸的校核：

1、模具厚度的校核：

模具厚度必需满足下式：

Hmm ≤ Hm ≤ Hmax

式中：Hmm——模具厚度

Hmin——注塑机允许最小模具厚度

Hmax——注塑机允许的最大模具厚度

2、模具外形尺寸的校核

1）注塑模外形尺寸应小于注塑机工作台面的有效尺寸。模具长宽方向的尺寸要与注塑机拉杆间距相顺应，模具至少有一个方向的尺寸能穿过拉杆间的空间装在注塑机的台面上。

2）模具与注塑机接触面积不能大小。由于接触面积大小，在高压下合模则可能使模具堕入模板，使模板或模具屈从变形碎裂，或在循环压力下疲倦断裂，因而应对模板与模具的接触压力停止强度核对。关于铸钢模板，平安许用压应力取55Mpa。普通按经历判别，接触面积不得小于最大有效接触面积的1/2

3、模具装置尺寸的校核

注塑机的动模板，定模板台面上有许多不同间距的螺钉或“T”型槽，用于装置固定模具，模具装置固定办法有两种：螺钉固定、压板固定，采用螺钉直接固定时（大型注塑机常用此法），模具动定模板上的螺孔及其间距必需与注塑机模板台面上对应的螺孔分歧，采用压板固定时（中、小模具多用此法），只需在模具的固定板左近有螺孔就行，有较大灵敏性。

4、喷嘴尺寸校核

注塑模主浇口凹坑的球面半径R应大于注塑机喷嘴球头半径r，常取R = r + (0.5-1)mm为宜,否则影响主浇道内凝料脱出。

而主浇道始端浇口直径D应大于喷嘴孔径d，通常取D = d + (0.5-1)mm,以利于塑料熔体的活动。：

三   开模行程和塑件推出间隔的校核

由于注塑机的开模行程是有限制的，取出制件所需求的开模间隔必需小于注塑机的最大开模间隔。开模行程的校核有三种状况：

1、注塑机的最大开模行程与模具厚度无关

如：液压——机械式锁模机构注塑机

1）关于单分型面注塑模具，其开模行程按下式校核：

S ≥ H1 + H2 +（5-10）mm                           3-（1）

式中S——注塑机最大开模行程mm

H1——塑件脱模间隔mm

H2——包括流道凝料在内的塑件高度mm

2）对双分型面注塑模，开模行程按下式校核：

S ≥ H1 + H2 + a +（5-10）mm                       3-（2）

式中a ——模具定模底板与定模型腔板分开的间隔mm

2、注塑机最大模行程与模具厚度有关

如：全液压式锁模机构的注塑机，此时最大开模行程等于注塑机动定模板的最大间隔SK减去模具厚离Hm.

1)关于单分型面模具，按下式校核：

S = SK – Hm ≥ H1 + H2 +（5-10）                3-（3）

SK ≥ Hm + H1 + H2 +（5-10）                     3-（4）

2）关于双分型面模具，按下式校核：

S = SK – Hm ≥ H1 + H2 +a（5-10）               3-（5）

SK ≥ Hm + H1 + H2 + a（5-10）                   3-（6）

3、有侧向抽芯模具的开模行程校核

当应用开模行程完成侧向抽芯时，开模行程的校核应思索完成抽拔距L1而所需求的开模行程HC

当HC > H1   +H2时，则按下式校核：

S ≥ HC +（5-10）

当HC ≤ H1 + H2时，仍按（3-（1））——（3-（6））式校核

此外，注塑机的顶出杆直径及双推杆中心距以及最大顶出间隔也应与模具相配。

因而，选择注塑机时，当工艺参数、模具外形尺寸、最大开模行程的校核有一条件不成立时，应选用另一级机型。而当装置固定尺寸、喷嘴与浇口、顶针的校核不成立时，购机时应联络供给商对机台停止部分改善，或自行改善模具以配合机器，要么选用上一级机型。
